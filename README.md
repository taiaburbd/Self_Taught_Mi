# Self_Taught_Mi
Self-Taught Medical Imaging (MI), Web base learning, quiz, and medical image store and visualized system. 

## Installation step
* git clone https://github.com/taiaburbd/Self_Taught_Mi.git
* pip install flask
* flask run


## Technologies
Project is created with: 
* Python 3.*
* Flask 
* Machine Learning 
* Deep Learing
